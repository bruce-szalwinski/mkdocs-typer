### goodbye

Simple program that says goodbye NAME for a total of COUNT times.

Usage:

```
cli bar goodbye [OPTIONS]
```

Options:

```
  --count INTEGER  Number of goodbyes.
  --name TEXT      The person to greet.  [required]
```
